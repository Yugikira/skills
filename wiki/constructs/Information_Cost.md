# Information Cost

## Definition
Cost c paid by informed traders to observe a realization of the signal θ.

## Mathematical Representation
c > 0 (positive cost), appears in γ(λ) = e^ac × √(Var(u|θ)/Var(u|w_λ))

## Role in Model
- Higher c → fewer informed traders (lower λ)
- Higher c → less informative price system (lower ρ_θ²)
- When c → 0, all traders informed; when c too high, none informed
